## 📖 О проекте

Интерактивный веб-тренажёр для проверки знаний по математическому анализу.  
Разработан для студентов МГУ (филиал в Душанбе) как инструмент самоподготовки к экзаменам.

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| ∫ Интегралы | 3 уровня сложности, 100+ задач |
| f'(x) Производные | 3 уровня сложности, 100+ задач |
| ∑ Ряды и последовательности | 3 уровня сложности, 100+ задач |
| ⏱ Таймер | Настраиваемое время: 15–30 минут |
| 📊 Статистика | История тестов, лучший результат, прогресс |
| 📚 Теория | Встроенные справочники по каждому разделу |
| 📱 Адаптивность | Работает на мобильных устройствах |
| 🔢 Формулы | Рендеринг LaTeX через MathJax |

---

## 🗂 Структура разделов

### ∫ Интегралы
- **Лёгкий** — табличные интегралы, линейные подстановки
- **Средний** — интегрирование по частям, тригонометрические подстановки
- **Сложный** — рациональные дроби, нестандартные приёмы

### f'(x) Производные
- **Лёгкий** — таблица производных, правило суммы
- **Средний** — произведение, частное, сложная функция
- **Сложный** — логарифмическое дифференцирование, многоуровневые цепочки

### ∑ Ряды и последовательности
- **Лёгкий** — понятие предела, геометрический ряд, необходимое условие
- **Средний** — признаки Даламбера, Коши, Лейбница, степенные ряды
- **Сложный** — равномерная сходимость, ряды Фурье, ряды Лорана

## 📊 Как работает тест

1. Введи имя → выбери раздел → выбери уровень сложности
2. Отвечай на 20 случайных вопросов за выбранное время
3. После теста — детальный разбор ошибок с правильными ответами
4. Результаты сохраняются в статистику автоматически
